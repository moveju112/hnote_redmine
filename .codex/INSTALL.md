# Installing hnote_redmine for Codex

Codex CLI에서 `hnote_redmine` 스킬을 사용하기 위한 설치 방법입니다.

## 설치

Codex에서 아래 명령을 실행합니다.

```text
$skill-installer https://github.com/moveju112/hnote_redmine
```

설치가 완료되면 Codex가 스킬을 사용할 수 있는 상태로 등록됩니다.

## 확인

Codex에서 스킬이 정상 인식되는지 확인하려면, `protocol-spec` 관련 작업을 요청해 보세요.

예시:

```text
프로토콜 명세서 작성해줘
```

## 업데이트

이미 설치된 경우에도 같은 명령으로 최신 상태로 다시 설치하면 됩니다.

```text
$skill-installer https://github.com/moveju112/hnote_redmine
```

## 제거

이 문서에서는 제거 절차를 별도로 다루지 않습니다. 사용 중인 Codex 환경의 스킬 관리 방식에 맞춰 삭제하세요.
